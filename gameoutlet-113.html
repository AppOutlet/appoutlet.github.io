GameOutlet 1.1.3

By messiaslima
May 29, 2023

This week we released version 1.1.3 with improvements in the game screen and a new feature "Wishlist".

The game details screen

There is some visual changes:

  • Now we have a toolbar with the "navigation back" icon, the game title and a option to add this game to de favories. This last part works with the new feature. "Wishlist"
  • The deals list is more clean and shows the "FREE" label when the game is free

Wishlist

We are introducing the wishlist in this release. A screen where the user can see its favorite games.¬†

Windows support

After some time of investigation, I finally was able to find the problem that prevents us from running a release version of GameOutlet in Windows hosts. The issues were already solved and we have a usable Windows version.

We also applied a change to make the Windows title bar integrated into the application background.

These were the updates for this version.
