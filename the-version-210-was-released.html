The version 2.1.0 was released!

By messiaslima
May 16, 2022

In this version we mainly released some theme and snaps support improvements.

System theme

Now, you can select the system theme on the theme list. This way, we can detect whether you are using a light or dark theme and show you the one that fits it.

Snaps are back in town!

We solved the snap indexing issue that we had an you are already are able to find any snap on App Outlet!

Minor improvements

  • Now we are caching the snaps data after the search
  • The links from patreon and open collective were replaced by the Ko-fi one

The detailed changelog can be found here

We hope you are enjoying these new changes and we see you in next one!
