GameOutlet 1.0.1

By messiaslima
May 2, 2023

The AppOutlet team is proud to announce that we have released the first version of the GameOutlet.

GameOutlet is a application that works like an aggregator of the best prices from the most popular PC games online stores.

Features

In this version you can

  • See the latests deals
  • See the deals from a game and be taken to the store page
  • Search by your favourite game

Future

In the future, we are planning to add more features like:

  • Wishlist
  • Alerts
  • Deals sharing

as well as an Windows version.

For more information, please see the project page in Github

Thank you!
