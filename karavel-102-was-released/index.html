🎊️ Karavel 1.0.2 was released!

By messiaslima
July 28, 2022
0

Karavel is one of our projects that is currently being developed. It is an simple and lightweight navigation library for desktop applications using Kotlin and Compose for desktop!

Now it is available on Maven Central.

The first version of Karavel was available only over Jitpack package registry. The version 1.0.2 (and the next ones) can be downloaded direcly from maven central repository.

Other improvements

Besides that the new release also has new versions of Kotlin and Compose dependencies.

You can checkout karavel documentation an code on Karavel github repository

If you are developing desktop applications with compose, please consider using Karavel to navigate between your composables!

See you next time! ⛵️
