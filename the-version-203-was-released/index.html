🎉 The App Outlet v2.0.3 was released!

By messiaslima
February 28, 2022
0

On the version 2.0.3, we have some improvements, dependency updates and some temporary changes regarding snap packages.

Features

🏡 Home menu item

Now, on side menu we have an option to take us to the application home

🇨🇱 Improving support to latin america spanish

Due to a collaboration of a supporter on the transifex, we improved the support to es-CL and es-419

Other

Update version of Angular and other dependencies

We updated the Angular and other dependencies. Angular was updated to version 13.

Snaps removal

Due to an update on snapcraft API, we had to remove the snaps from the store. We are already working on a solution

the detailed change log can be found here
